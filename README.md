
## 🙌 Nucleotide Sequence Analyzer (NSA)

## ❓ NSA 가 뭐에요?   
- **NSA**는 MSA 를 적용한 염기서열 분석 프로젝트입니다!

## 🙋‍♀️ NSA-MSA
